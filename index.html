Задание №1

ssh (Secure Shell) используется для безопасного удалённого доступа к серверу. С его помощью можнно выполнять команды на сервере удалённо, будто вы работаете на нём локально. На скриншоте показано подключение к учебному серверу kubsu-dev.ru с использованием личных логина и пароля.

Команда ping используется для проверки досупности удалённого сервера в сети. Она отправляет ICMP-запросы и показывает IP-адрес хоста, время хостинга. В данном случае с помощью ping определён IP-адрес сервера kubsu.ru

Команда nslookup используется для получения DNS-домена, показывая соответсвия доменного имени его IP-адресу.

MX-записи указывают почтовые серверы, обслживающие домен. С помощью команды nslookup -type=mx были определены серверы, ответственные за приём эл. почты.

Команда whois нужна для получения регистрационной информации о домене. В данном случае была определена дата регестрация и сведения о владельце домена.
